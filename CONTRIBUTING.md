# Contributing to MolSysSuite Talks

## Add a new talk

1. Create: `talks/<year>-<slug>/`
2. Add `README.md` using `templates/talk-README.md`.
3. Add `outline.md` and `objectives.md`.
4. Add slide notes in `notes/`.
5. If videos are used, store them externally and document links in `media/README.md`.
6. Update the corresponding tool repository pointer (`resources/talks.yml`).

## Naming

Prefer: `YYYY-<short-slug>`.
