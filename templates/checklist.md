# Talk checklist

## Content
- [ ] Outline and timing
- [ ] Slide titles reflect the narrative
- [ ] A single “technical foundations” slide (if relevant)
- [ ] Clear closing + call to action

## Demos
- [ ] Prefer recorded demos over live demos
- [ ] Videos tested on the presentation laptop
- [ ] Backup PDF exported

## Delivery
- [ ] Local copy of slides + media
- [ ] Optional USB backup
